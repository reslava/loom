# VSCode Loom Explorer toggle

## Problem

Sometimes we need to use VSCode `Explorer` to do file operations. Currently Loom always sync opened loom documents with tree. This could be annoying in some scenarios.

## Idea

Toggle toolbar button to enable/disable (use 2 codeicon lib icons):
Loom opened doc -> sync tree

Other sync should be always active:
Tree node -> open loom doc
