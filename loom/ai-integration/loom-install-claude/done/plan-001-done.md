## Step 2 — Rework the root-CLAUDE.md patch so both @.loom/CLAUDE.md and @CLAUDE-LOCAL.md import lines are present at the top, idempotently — handle file-missing, one-present, and both-present.

Reworked the root-`CLAUDE.md` patch (former step 3) to ensure BOTH imports idempotently. Replaced the single `importLine` logic with `loomImport`/`localImport` and a `missing = [..].filter(line => !existing.includes(line))` computation: prepend only the missing lines (Loom contract first), create the file with both lines when absent, and no-op (rootClaudeMdPatched=false) when both are already present. Per-line substring guards mean a re-run never duplicates and a file with one import gains only the other.
