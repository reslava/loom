# Connect an AI (optional)

The tree, documents, and all structure work with **no AI at all**. AI is only needed for the *generate / refine / do-step* actions.

**Recommended — Claude Code (no API key).** If the `claude` CLI is on your PATH, Loom's AI buttons open a terminal and run an agent you can watch and steer, using your existing Claude subscription.

```
npm install -g @anthropic-ai/claude-code
```

**Fallback — API key.** No Claude Code? Set an API key in Settings (`reslava-loom.ai.apiKey`) and Loom uses the provider directly.

The **AI status bar item** shows which path is active (Claude Code / API key / not set up); click it any time to set one up.
