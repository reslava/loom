# Create your first weave

A **weave** is a project area; inside it, **threads** are workstreams. A thread holds the documents that carry a piece of work from idea to done:

```
weave → thread → idea → design → req → plan → done
```

Two ways to start:

- **Create Weave** — create an empty weave and build it up yourself.
- **Start with an example** — seed one tiny, clearly-labelled `example` weave so you can see the whole shape (idea → plan → steps) immediately. Delete it whenever you like.

Everything you create appears in the **Loom** panel in the Activity Bar.
